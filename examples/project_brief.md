# 项目需求说明

项目名称：AI Agent 辅助嵌入式研发与测试分析系统

项目背景：
在嵌入式系统课程设计、论文写作和 CAN 诊断测试中，需求拆解、代码结构设计、测试日志分析和文档整理通常需要大量人工处理。  
本项目希望构建一个轻量级 Agent 工作流，用于辅助完成工程项目从需求到报告输出的闭环。

核心需求：
1. 支持将项目需求拆解为功能模块、接口约束和验收指标。
2. 支持生成嵌入式代码开发任务，例如 STM32、ESP32、CAN、MQTT、INA226、OLED 等模块。
3. 支持解析 CANoe 测试日志，提取请求、响应、测试结果和异常原因。
4. 支持生成测试结果 CSV 表格。
5. 支持生成 Markdown 项目报告，便于后续整理到 Word 或 PPT。

目标成果：
- Agent 工作流代码
- 测试日志分析结果
- 项目报告
- 可上传至 GitHub 的成果展示仓库
